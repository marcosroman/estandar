Planos Estandar

Esta app web sirve para generar planos compuestos por planos estandar.

Para hacer uso de un plano estandar, este debe haber sido previamente cargado.

¿Cómo cargar un código estandar?

Si un código estandar todavía no fue cargado, podés agregarlo ingresando al menú ≡ Estandar,
seleccionando Cargar Nuevo Estandar e introduciendo los datos del catálogo.

  1. Se ingresa el código estandar y se escoge la categoría.
  2. Para la imagen,
    1. seleccionamos la del catálogo,
    2. copiamos (seleccionamos la imagen solamente, sin el nombre ni los comentarios) con ctrl+c o usando el menu,
    3. luego pegamos (con ctrl+v).
  3. Por último, click en 'Guardar'

y ya tenemos el nuevo plano estandar agregado.

(Se pueden agregar tambien los comentarios que figuran en el catálogo, una vez cargado.)

Los planos estandar que ya fueron cargados, se pueden acceder mediante el menu, siguiendo el enlace Planos Estandar

Generar un nuevo plano (compuesto por planos estandar)

Para crear tus propios planos, elegí Generar Nuevo Plano en el menú, donde ingresás el OT o Nro. de Venta, seguido de los detalles del mismo.

Tras guardar, el plano se genera automáticamente.

Detalles para la carga

Para ver la lista de planos que ya generaste, utilizá la opción Mis planos del menú.
